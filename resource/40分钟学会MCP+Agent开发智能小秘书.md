# 40分钟学会MCP+Agent开发智能小秘书

#### 今晚：20:00开始



#### 项目功能需求：

- 有完整的操作界面，并支持流式输出。
- 可以获取网络上公开的数据。
- 可以解析和提取网页内容。
- 可以帮我自动分析数据，并生成分析图表。
- 可以帮我查询火车票，机票等。
- 可以规划出行路线。
- 可以帮我规划菜单和推荐减肥菜谱。
- 等等。。。

#### 项目技术架构：

LangGraph==0.6.3  +  LangChain==0.3.68  +  MCP + Gradio==5.39 + Python==3.12

#### 项目开发步骤：

1. 创建项目，并准备大模型的环境变量；
2. 配置和部署MCP服务端；
3. 配置和创建MCP的客户端；
4. 从MCP的客户端获取Tool列表，并创建Agent；
5. 运行和测试Agent；
6. 开发智能助手的界面并部署Agent；
